# AliyunSVideoDemo

### 1.新建ionic项目
    ionic start

### 2.添加Android平台
    ionic cordova prepare android

### 3.添加插件
    ionic cordova plugin add https://github.com/hellomooning/AliyunSVideoPro.git

### 4.调用
#### 视频拍摄
    cordova.plugins.AliyunSVideoPro.startRecord();

#### 视频剪辑
    cordova.plugins.AliyunSVideoPro.startCrop();
